1. For create library call "python CreateLibrary.py lib_name namespace_name" or
"CreateLibrary.bat lib_name namespace_name" for windows

if lib_name == MyLib and namespace_name == MyNMSP then Mynmsp-MyLib folder and
MyLib library class will be created.

2. For create component call "python CreateComponent.py lib_name namespace_name component_name [inheritance_component_name]"
or "CreateComponent.bat lib_name namespace_name component_name [inheritance_component_name]" for windows

lib_name should be full library folder name (include namespace_name if exist, e.g.
Mynmsp-MyLib).
